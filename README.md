# HR-Employee-Data-Integrity-Dashboard
HR Employee Data Integrity, Validation, Reconciliation &amp; Audit Dashboard built using Excel

This project demonstrates cleaning, validation, reconciliation, and audit of HR employee data using Excel.

### Tools Used
- Microsoft Excel
- Power Query
- Data Validation
- Pivot Charts
- COUNTIF & IF formulas

### Key Highlights
- Total Employees: 1470
- Audit Passed: 100%
- Review Required: 0
